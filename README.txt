******* FR **********
TP Cobol : application de gestion de chauffeurs de bus

Programme cobol utililsant l'accès aux fichiers de données 
(lecture, recherche, écriture, suppression) 
Le fichier "menu" affiche le menu (étonnant)
et donne accès à : 
  - la gestion des chauffeurs
  - la gestion des affectations
  - les disponibilités bus / chauffeurs / dates 
  


******** ENG ***************** 
Cobol project : application to manage bus drivers

This program uses external data files (read, search, write, delete). 
The "menu" file displays the menu (how great) and gives access to : 
  - drivers' management
  - allocations' management
  - bus / drivers / dates availabilities
